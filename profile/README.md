# Turnberry Manor Condominium Association

The Turnberry Manor Condominium Association ("TMCA") is an Illinois not-for-profit corporation governing a residential condominium community of **344 units in 51 buildings** in Roselle, Illinois (DuPage County).

This GitHub organization holds the source for the Association's working documents and public-facing website, maintained by a Director on a volunteer basis. The Board of Directors does not see the repositories directly — only rendered deliverables (Word documents, PDFs, the published website).

## Repositories

- **[tmca-docs](https://github.com/turnberry-manor/tmca-docs)** — Source for the Amended and Restated Rules and Regulations (TMCA Restatement 2026). Builds to DOCX, PDF, and AcroForm fillable PDFs.
- **[tmca-site](https://github.com/turnberry-manor/tmca-site)** — Source for the public information website (Hugo, deploys to Cloudflare Pages).
